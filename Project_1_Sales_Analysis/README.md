# Sales Performance Analysis (Revenue Focused)

## Problem Statement
The objective of this project is to analyze sales data to understand overall performance, identify trends, and provide actionable business insights.

## Business Questions
- What is the overall sales performance?
- Which categories and sub-categories generate the most sales?
- Which regions and cities contribute the highest sales?
- How do sales trends change over time?
- Which customer segments drive the most revenue?

## Tools Used
- Excel

## Dataset
The dataset contains order-level sales data including Order ID, Order Date, Ship Date, Ship Mode, Customer ID, Customer Name, Segment, Country, City, State, Postal Code, Region, Product ID, Category, Sub-Category, Product Name, and Sales.
